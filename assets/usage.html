Logger

Usage:

GET /log

Log a request. Parameters:

  • api_key

All the additional parameters are going to be recorded in the document.

GET /pixel.png

Log a request and server a pixel. The image_url parameter can override the default 1x1 gif pixel. Parameters:

  • api_key

All the additional parameters are going to be recorded in the document.

Encryption

All the data can be sent as a stringified and base64 encoded JSON using the parameter e. Example:

/log?e=eyJhcGlfa2V5IjoxMjMsIm5hbWUiOiJkYW5pZWxlIn0=

Where the encoded string is equivalent to 

{api_key:123,name:'daniele'}
